# 大文字・小文字 行ごと判定ツール (Case Detector by Line)

入力されたテキストを1行ずつ解析し、その行が「すべて大文字」「すべて小文字」「大文字小文字が混在」、あるいは「アルファベットなし」のどれに該当するかを判定するWebアプリケーションです。

## 特徴

- **スペース・記号・数字の自動無視**: 行内にスペース、数字、記号（`!`, `?`など）が含まれていても、純粋なアルファベット部分だけを抽出して正確に判定します。
- **入力値の保持**: 判定ボタンを押した後も、テキストエリアに入力した文字が消えずに残るため、連続して編集・判定が可能です。
- **マルチプラットフォーム対応**: Windows、Mac、LinuxなどのOSによる改行コードの違い（`\r\n` や `\n`）を自動で判別して処理します。
- **安全設計**: クロスサイトスクリプティング（XSS）を防ぐため、サニタイズ（エスケープ処理）を行っています。

## 動作環境

- PHP 7.0 以上（`ctype` 拡張機能が有効であること ※通常のPHP環境では標準で有効です）
- 各種Webブラウザ（Chrome, Safari, Edge, Firefoxなど）

## 設置・利用方法

1. **ファイルの配置**
   本リポジトリの `index.php` を、PHPが動作するWebサーバー（Apache、Nginx、XAMPP、MAMP、Docker環境など）の公開ディレクトリ（`htdocs` や `www` など）に配置します。

2. **ブラウザでアクセス**
   サーバーを起動し、ブラウザから該当のURL（例: `http://localhost/index.php`）にアクセスします。

3. **ツールの使用**
   - 画面上のテキストエリアに、判定したい文章を入力（またはペースト）します。
   - 「判定する」ボタンをクリックします。
   - 画面下部に、行番号ごとの判定結果が表示されます。

## 判定仕様の例

| 入力行の例 | 判定結果 | 理由 |
| :--- | :--- | :--- |
| `HELLO WORLD!!` | すべて【大文字】です。 | スペースや `!` を除いたアルファベットがすべて大文字のため |
| `this is a pen.` | すべて【小文字】です。 | スペースや `.` を除いたアルファベットがすべて小文字のため |
| `Php 8.4` | 【大文字と小文字が混ざって】います。 | `P` が大文字、`hp` が小文字のため（数字は無視） |
| `12345 (!!!)` | アルファベットが含まれていません。 | 判定対象となるアルファベットが存在しないため |
